# AppKit Reference

The AppKit reference lives in versioned docs synced from `databricks/appkit`.

- AppKit current docs entrypoint: `/docs/appkit/current`
- AppKit v0 channel (latest v0.x.y): `/docs/appkit/v0`

`/docs/appkit/current` is intentionally a pointer page so the docs content itself is not duplicated.

Use this page to locate:

- current AppKit API reference for your deployed version
- migration notes when moving between AppKit versions
- canonical examples for plugin contracts and runtime integration points
