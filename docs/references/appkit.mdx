## How to navigate

- Use **References > AppKit** in the docs sidebar to expand versions.
- Select a major channel (`Current`, `v0`, and future majors as they are synced).
- Expand a channel to browse full docs and API pages.

## Why this structure

- one stable AppKit entry point in `References`
- progressive expansion for large doc trees
- fast version switching on mobile and desktop
