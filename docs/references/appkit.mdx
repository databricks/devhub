# AppKit Reference

AppKit reference docs now open directly from **References > AppKit** in the docs sidebar.

If you landed on this URL directly, continue to the latest synced channel:

- [Open AppKit v0 (current) channel](/docs/appkit/v0)

## How to navigate

- Open **References > AppKit** to jump into the latest docs page.
- Use the AppKit sidebar version picker to switch channels.
- Use **Back to main docs** in the AppKit sidebar to return to the main docs navigation.

## Why this structure

- faster entry into the docs users actually need
- a focused AppKit-only sidebar while reading AppKit docs
- one-click return to the broader docs navigation
